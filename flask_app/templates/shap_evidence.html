SHAP Evidence

This page presents SHAP as a supporting explainability method used during the experimental stage of the DermaAI system. Grad CAM remains the primary real time explanation method in the deployed Flask prototype.

What SHAP shows

SHAP explains how different image regions contribute to a model prediction. In this project, it was used to observe pixel level contribution patterns for a selected dermoscopic image and compare them with the model prediction output.

Why it is supporting evidence

SHAP can be computationally expensive for image based deep learning models. Therefore, it was implemented as a notebook based explainability experiment, while Grad CAM was selected for real time use in the Flask application.

Academic interpretation

SHAP was implemented to strengthen the explainability component of the research prototype. However, because the system is designed as a lightweight Flask application, the final deployed workflow uses Grad CAM for real time visual explanation. This design choice balances transparency, usability and computational feasibility.
