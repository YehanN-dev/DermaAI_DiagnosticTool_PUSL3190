User guidance

Help

Use this page to understand how to upload an image and interpret the system output safely.

Supported images

The system accepts JPG, JPEG and PNG dermoscopic image files up to 8 MB.

How to analyse

Select or drag an image into the upload area, then click Analyse Image. The system will generate a prediction report and Grad CAM output.

How to read confidence

A higher confidence score means the model assigned more probability to the predicted class. It does not guarantee clinical correctness.

Important warning

This application is a research prototype. Always consult a qualified healthcare professional for medical diagnosis and treatment decisions.
