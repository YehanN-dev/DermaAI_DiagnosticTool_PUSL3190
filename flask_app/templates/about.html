About the system

DermaAI Diagnostic Support Tool

DermaAI is an academic research prototype designed to support skin lesion analysis using deep learning and explainable AI. The system classifies dermoscopic images into HAM10000 lesion categories and provides Grad CAM visualisation to improve model transparency.

Purpose

The purpose of this project is to demonstrate how a trained image classification model can be integrated into a usable diagnostic support interface. It is designed for research and learning, not for independent medical diagnosis.

Model

The implemented model is based on fine tuned MobileNetV2. It produces probability scores across seven HAM10000 skin lesion classes and displays the most likely prediction.

Explainability

Grad CAM is used to highlight the image regions that influenced the model output. This helps users understand which visual areas contributed to the prediction.

Limitations

This tool does not replace dermatologist assessment, biopsy, clinical history or professional medical judgement. All outputs must be treated as decision support only.
