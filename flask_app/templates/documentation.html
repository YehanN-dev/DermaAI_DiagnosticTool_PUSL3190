System workflow

Documentation

This page explains the main workflow followed by the DermaAI diagnostic support prototype.

1

Image Upload

The user uploads a dermoscopic skin lesion image in JPG, JPEG or PNG format.

2

Preprocessing

The image is resized to the model input size and preprocessed using the MobileNetV2 preprocessing pipeline.

3

Prediction

The fine tuned MobileNetV2 model generates probability scores for seven HAM10000 lesion classes.

4

Decision Support

The system displays the primary prediction, top probability scores and confidence based review flags.

5

Grad CAM Explanation

Grad CAM highlights the image regions that contributed most strongly to the model prediction.
